01

Pengenalan Aplikasi E-Company

Sistem manajemen SDM berbasis desktop untuk mempermudah pengelolaan karyawan dan penggajian.

Apa itu E-Company?

E-Company adalah aplikasi desktop manajemen perusahaan yang dirancang untuk membantu tim HR dan Admin dalam mengelola data karyawan, menghitung gaji secara otomatis, memantau status pembayaran gaji, serta mengelola pengajuan cuti dan izin karyawan. Aplikasi ini dibangun menggunakan teknologi JavaFX dengan database relasional.

Persyaratan Sistem

Kebutuhan Perangkat

  • Sistem Operasi: Windows 10/11, macOS, atau Linux
  • Java Runtime Environment (JRE) versi 11 atau lebih baru
  • RAM minimal 2 GB (4 GB direkomendasikan)
  • Ruang disk minimal 200 MB
  • Resolusi layar minimal 1280 × 720

Kebutuhan Database

  • MySQL Server versi 5.7 atau lebih baru
  • Koneksi database aktif sebelum menjalankan aplikasi
  • Konfigurasi host, port, user & password di file config
  • Database yang telah diimpor dengan skema awal

Login & Register Perusahaan

Proses masuk ke sistem dan pendaftaran perusahaan baru.

Pengelola penuh seluruh fitur sistem

  • Kelola semua data karyawan

Cara Login ke Aplikasi

1
Buka Aplikasi E-Company

Jalankan file aplikasi (.jar) atau shortcut di desktop. Halaman login akan muncul secara otomatis.

2
Masukkan Username & Password

Ketik username dan password yang telah didaftarkan oleh admin perusahaan.

3
Pilih Role / Peran

Klik dropdown "Login Sebagai..." dan pilih peran: Admin atau Employee. Pastikan role sesuai dengan akun Anda.

4
Klik Tombol LOGIN

Sistem akan memverifikasi data dan mengarahkan ke dashboard sesuai role yang dipilih.

05

Penggajian (Payroll)

Proses perhitungan dan pembayaran gaji karyawan dengan kalkulasi otomatis.

Tentang Modul Payroll

Modul ini memungkinkan Admin menghitung gaji bersih karyawan secara otomatis berdasarkan gaji pokok, tunjangan tambahan, uang lembur, dikurangi potongan BPJS Kesehatan (3%) dan pajak PPh 21 yang dihitung secara progresif.

Langkah Proses Penggajian

1
Buka Menu "💸 Penggajian (Payroll)"

Klik menu di sidebar kiri. Daftar semua karyawan akan tampil di panel kiri.

2
Pilih Karyawan dari Daftar

Klik nama karyawan di panel kiri. Sistem otomatis mengisi gaji pokok dan nomor rekening.

3
Masukkan Tunjangan & Lembur (Opsional)

Isi field "Tunjangan Tambahan" dan "Uang Lembur" jika ada. Total akan dihitung secara otomatis saat mengetik.

4
Verifikasi Rincian Gaji

Periksa kalkulasi potongan BPJS, PPh 21, dan Total Take Home Pay yang muncul.

5
Klik "✔ Bayar & Simpan Data"

Data pembayaran tersimpan ke database dan status karyawan berubah menjadi "Sudah Dibayar".

Formula Perhitungan Gaji

Gaji Pokok + Rp 6.500.000
Tunjangan Tambahan + Rp 500.000
Uang Lembur + Rp 250.000
Potongan BPJS Kesehatan (3%) − Rp 195.000
Potongan PPh 21 (Progresif) − Rp 125.000
TOTAL TAKE HOME PAY = Rp 6.930.000

Potongan BPJS (3%)

BPJS Kesehatan dipotong sebesar 3% dari gaji pokok. Potongan ini dihitung otomatis dan tidak bisa diubah secara manual.

BPJS = Gaji Pokok × 3%

Pajak PPh 21

PPh 21 dihitung secara progresif sesuai aturan perpajakan Indonesia berdasarkan penghasilan kena pajak (PKP) tahunan karyawan.

  • PKP ≤ 60 juta → 5%
  • 60–250 juta → 15%
  • 250–500 juta → 25%
  • > 500 juta → 30%

FAQ

Pertanyaan yang Sering Diajukan

Solusi untuk masalah umum yang ditemui pengguna.

❓ Login gagal meskipun password benar

Pastikan Role yang dipilih di dropdown sesuai dengan role akun Anda di database. Akun Admin tidak bisa login dengan role Employee, dan sebaliknya. Juga pastikan koneksi database aktif.

❓ Gaji karyawan sudah dibayar tapi status masih "Belum Dibayar"

Pastikan proses "Bayar & Simpan Data" benar-benar di-klik dan tidak ada pesan error. Status diperbarui real-time — coba refresh dengan berpindah menu lalu kembali ke Dashboard.

❓ Potongan PPh 21 terasa terlalu besar atau kecil

PPh 21 dihitung berdasarkan penghasilan kena pajak tahunan. Karyawan dengan gaji di atas PTKP (Penghasilan Tidak Kena Pajak) akan dikenakan tarif progresif. Pastikan data gaji pokok sudah benar.

❓ Karyawan tidak muncul di daftar Payroll

Pastikan data karyawan sudah ditambahkan melalui menu "Data Karyawan" dan informasi lengkap termasuk gaji pokok dan nomor rekening sudah terisi.

❓ Aplikasi tidak bisa terhubung ke database

Periksa konfigurasi koneksi database (host, port, username, password) di file konfigurasi aplikasi. Pastikan MySQL Server berjalan dan port tidak diblokir firewall.
